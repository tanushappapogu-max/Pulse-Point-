# Pulse Point

Pulse Point is an early prototype for SSE, Synchronous Spatial Echo: a haptic-first assistive spatial interface for blind and low-vision users.

Instead of adding more spoken directions, Pulse Point explores how a phone and optional haptic ring can translate nearby space into touch: object search, social attention shifts, ambient obstacle alerts, and liquid-level guidance.

## Project

The starter app lives in [`pulse-point`](./pulse-point).

```bash
cd pulse-point
npm install
npm run dev
```

## Prototype Goals

- Preserve hearing by keeping the primary interface visual/tactile instead of audio-heavy.
- Demonstrate four early SSE modes: Pinpoint Search, Social Gaze Map, Ambient Obstacle Filter, and Liquid Level.
- Visualize a 3x3 haptic matrix ring vocabulary for directional and proximity signals.
- Provide a clean foundation for a TSA concept demo, pitch deck, or future hardware/software prototype.
